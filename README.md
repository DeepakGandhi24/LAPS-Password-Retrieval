🔐 LAPS Password Retrieval

A macOS application designed to securely retrieve Local Administrator Password Solution (LAPS) credentials for managed devices in enterprise environments.

📌 Overview

This tool provides a streamlined and secure interface for IT administrators to fetch LAPS passwords without needing to manually query directory services or run scripts.

It is built using Swift and macOS native frameworks, ensuring a smooth, modern, and secure user experience.

✨ Features
🔍 Retrieve LAPS passwords for devices
🔐 Secure handling of credentials
🖥️ Native macOS UI (Swift / SwiftUI / AppKit)
⚡ Fast and responsive interface
🧩 Designed for enterprise environments
📋 Easy-to-use interface for IT admins
